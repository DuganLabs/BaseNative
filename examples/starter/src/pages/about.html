Design principles

No namespace theater
Semantic HTML only. W3C primitives are the component model.
Zero inline styles
All CSS lives in cascade layers. No style="", no CSS-in-JS.
Trinity Standard
State, logic, and template fuse in one file.
No build step
Native ESM. No bundler required in development.
